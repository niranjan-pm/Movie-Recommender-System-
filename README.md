# Movie-recommendation-system
• Preprocessed a 44K+ movie dataset and generated embeddings using Transformers’ AutoTokenizer for training 
• Utilized Hugging Face’s BERT-base-uncased model to transform descriptions into meaningful text vector embeddings 
• Implemented cosine similarity and built a Gradio app, efficiently delivering top-5 personalized movie recommendations 
The dataset is here: https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset?resource=download
